Aplicação criada como desafio do curso FrontEnd da Rocketseat!
Skills Utilizadas:
HTML
CSS
StyledComponent
Typescript
ReactJS
Hoocks
JSON-Server
Desafio bem completo para apresentar diversos conhecimentos adiquiridos

Para iniciar o "banco de coffees"(JSON-SERVER): yarn server
